# graph
calculating in degrees and degree distribution of nodes in a graph

Python dictionary for directed graph GRAH0, GRAPH1, GRAPH2
function compute_in_degrees(graph) - computes the total numbers of indegrees of each node
function in_degree_distribution(graph) - computes the total number of degree distribution for a particular graph
make_complete_graph - computes all the possible degree distribution for a particular node without self looping (n - 1)
